# 战略家开源系统
战略家是一个开源的问答系统，基于keystoneJS 4.x系列开发。

keystoneJS 是nodeJS + express + mongodb 开发而成的CMS系统。

我们在 在此基础上先扩展admin系统，采用adminlte + vue + form-create，
完成了models到form的自动生成。
